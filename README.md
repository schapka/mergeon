# Mergeon

## CLI

```text
mergeon [entry]

Load JSON file

Commands:
  mergeon load [entry]  Load JSON file                                  [default]

Options:
  --version        Show version number                                 [boolean]
  --entry, -i, -e  Entry file                                [string] [required]
  --help           Show help                                           [boolean]
```
